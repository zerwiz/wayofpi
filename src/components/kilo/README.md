# Kilo Module - Minimal Display Component

## Installation
Already built and configured at:
- `src/components/kilo/index.ts`
- `src/components/Kilo.tsx`
- `src/types/kilo/types.ts`

## Usage
```typescript
import { Kilo } from '@kilo';

< Killo />
< Kilo height="100%" >Display Content</Kilo>
```

## File Structure
```
src/
  - components/
    - kilo/
      - index.ts          # Module entry point
      - components/
        - Kilo.tsx        # Kilo component
        - Kilo.css        # Styles
```
