# connect telegram

## Goal
- Add Telegram bot integration to Way of Pi for real-time notifications and user interaction.

## Assumptions & constraints
- Telegram bot token provided by the user (env variable TELEGRAM_BOT_TOKEN)
- Bot must be able to receive messages and send messages via Bot API.
- Integration should use the existing Pi extension system (`.pi/extensions/`) and be available as a tool command `telegram-notify`.
- Network connectivity required for HTTPS calls to Telegram API.
- Must respect rate limits (30 messages/second for standard bot).

### Phase 1: Project scaffolding (15 min)
1. Create directory `extensions/telegram-notify`.
2. Add shim: echo "export { default } from \"../../extensions/telegram-notify\"" > .pi/extensions/telegram-notify.ts
3. Add tool entry: echo "module.exports = async function(telegramNotify) { return require('child_process').execSync('node .pi/tools/telegram-notify.js \'+\' + JSON.stringify(telegramNotify)).toString(); }" > .pi/tools/telegram-notify.js
4. Initialize `package.json` for the tool (minimal, name `@pi/telegram-notify`, dependency `node-fetch`).

### Phase 2: Core telebot wrapper (`telegram.ts`)
- Load `TELEGRAM_BOT_TOKEN` from `process.env`.
- Implement `async function sendTelegramMessage(text: string): Promise<string>` using `fetch` to `https://api.telegram.org/bot${token}/sendMessage` with `chat_id` (configured via env `TELEGRAM_CHAT_ID`).
- Implement retry logic (max 3 attempts, exponential backoff).
- Return Telegram message ID or error.

### Phase 3: CLI tool (`commands.ts`)
- Define command `telegram-notify` with description "Send a notification to the configured Telegram chat".
- Parse arguments: `--message "Your text"` (required), `--chat-id` optional (uses env).
- Call `telegramNotify` and log success or error with colour.

### Phase 4: Config handling (`config.ts`)
- Export `const config = { token: process.env.TELEGRAM_BOT_TOKEN, chatId: process.env.TELEGRAM_CHAT_ID }`.
- Validate presence; throw clear error if missing.

### Phase 5: Bootstrap script (`scripts/setup-telegram.sh`)
```bash

# Install fetch polyfill if using node <18
```
