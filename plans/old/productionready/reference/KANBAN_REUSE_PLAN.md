## Implementation Steps

### Phase 1: Copy & Reskin (1-2 days)
1. Copy kanban components to `apps/wayofwork-ui/src/components/work/kanban/`
2. Global find/replace colors (use sed or VS Code batch replace)
3. Update imports to use Way of Pi's hooks (`useUiMode`, `useServerConfig`)
4. Test rendering in Work Mode

### Phase 2: Connect to Real APIs (1 day)
1. Replace mock services with `/api/portal/*` calls
2. Add JWT token to all API requests (already done in `api/client.ts`)
3. Test with real data from SQLite DB

### Phase 3: AI Prediction Integration (2-3 days)
1. Add planning document parser (`plans/PRD*.md`, `plans/API_SPECS.md`)
2. Create `useAIPredictions` hook that calls Pi with context
3. Add prediction UI to ClawSchedule views
4. Implement daily digest generation via `@WorkLeaderClaw`

1. `@WorkTimeBot`: Workers log time via "log 4.5h on Task #1"
2. `@WorkLeaderClaw`: Leaders query "What's behind schedule?" → Pi analyzes and responds
3. Timed messages: Daily 8 AM digest to leaders, weekly Friday summary
