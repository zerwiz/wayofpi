# Work Button Responsive View Specifications

## Mobile View (< 768px)

- **Button**: Large touch-friendly button (min 44px height)
- **Layout**: Single column
- **Projects**: Hide project name, show only hours
- **Status**: Show status color only

## Tablet View (768px - 1024px)

- **Button**: Medium-sized
- **Layout**: Two columns if space permits
- **Projects**: Show first project

## Desktop View (> 1024px)

- **Button**: Compact
- **Layout**: Multi-column grid
- **Projects**: Show all projects
- **Status**: Show status with icon

## Implementation

Each view has corresponding styles and layout variations for the work button component.
