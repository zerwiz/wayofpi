# 🗁 Session Agent

**Agent that knows all about chat sessions from `/home/zerwiz/piwithstuff/.pi/agent/sessions`**

---

## 🎯 Capabilities

| Feature | Description |
|---------|-------------|
| **List Sessions** | List all sessions across all workspaces |
| **Search Sessions** | Search by content, timestamp, or keywords |
| **Get Session** | Retrieve full session details and message history |
| **Export Sessions** | Export to JSON, TXT, or CSV formats |
| **Delete/Archive** | Remove or archive old sessions |
| **Stats** | Get session statistics and counts |

## 🛠️ Tools

### `list_sessions(filters)`
```typescript
await session_agent.list_sessions({
  workspace_filter: "--home-zerwiz-.pi--",
  limit: 100,
  search_query: "optional"
});
```

### `search_sessions(query, filters)`
```typescript
const results = await session_agent.search_sessions({
  query: "read file",
  workspace: "--home-zerwiz-.pi--",
  limit: 50
});
```

### `get_session(session_id)`
```typescript
const session = await session_agent.get_session({
  sessionId: "019d9cee-c669-77e2-8c2a-ca4c9cda34a1"
});
```

### `export_sessions(ids, format, path)`
```typescript
await session_agent.export_sessions({
  sessionIds: ["019d9cee-c669-77e2-8c2a-ca..."],
  outputFormat: "json",
  outputPath: "/tmp/sessions.json"
});
```

### `delete_session(id)`
```typescript
await session_agent.delete_session({
  sessionId: "019d9cee-c669-77e2-8c2a-ca4c9cda34a1"
});
```

## 📊 Session Message Types

- `session` - New session created
- `model_change` - Model switching event
- `thinking_level_change` - Thinking level update
- `message` - User or assistant message
- `toolCall` - Tool execution request
- `toolResult` - Tool execution result
